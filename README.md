Slash is a PHP framework designed to make creating dynamic web content easy.

## Installation

### Using Composer:

```sh
composer require archon/slash
```

```json
{
    "require": {
        "archon/slash": "0.0.0"
    }
}
```

### Requirements
 - PHP 5.3 or higher

### Dependencies
 - [gajus/dindent](https://github.com/gajus/dindent): 2.0.2
